Reflection

A look into what I learned during my first semester as a UMich UX graduate student.

  • SI 500 — Problem Solving with People, Information, & Technology

    In SI 500, I learned how to plan and conduct user interviews, analyze qualitative data, and translate messy notes into clear insights. This course helped me build a strong foundation in research methods and user-centered thinking.

    Skills: interviewing, affinity mapping, insight generation

  • SI 506 — Programming I

    SI 506 introduced me to Python by focusing on writing clean, efficient code. I worked on data transformations, dictionary-based structures, and automation logic. This course strengthened the technical skills that support my UX workflow.

    Skills: Python, data structures, automation, problem-solving

  • SI 539 — Web Design: Responsiveness and Accessibility

    In SI 539, I learned how to build accessible and semantic websites using HTML, CSS, and JavaScript. I created responsive layouts, practiced clean structure, and developed a stronger understanding of inclusive web design principles.

    Skills: HTML, CSS, JavaScript, responsive design, accessibility

  • SI 588 — Foundations of UX

    SI 588 introduced core UX frameworks, heuristics, and design principles. I explored how good UX supports user goals, reduces friction, and creates meaningful experiences. This course tied together theory and hands-on practice.

    Skills: UX heuristics, sketching, wireframing, evaluating usability
